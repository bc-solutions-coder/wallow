## Quick Start

Say any of these to Claude:
- **"Let's start the audit"** — begins at Phase 1, File #1
- **"Let's do Phase 5"** — jumps to Phase 5
- **"Let's do file #23 in Phase 4"** — jumps to specific file
- **"Where did we leave off?"** — Claude checks INDEX.md and phase docs to find your place
- **"Batch mode"** — review 3-5 small files at a time instead of one-by-one
- **"Skip this file"** — marks `[~]` and moves on
- **"Come back to this"** — marks `[?]` and moves on
